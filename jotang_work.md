## 偏导数、链式法则、梯度、矩阵等数学概念在机器学习中的作用
### 偏导数
1. 偏导数本来是在多变量函数中固定其余变量，对其中某一个变量进行求导，衡量其对于某一个特定变量的变化率，将多变量函数问题简单化。
2. 在机器学习中，偏导数通过对非固定的变量不断地进行计算，求出应该如何调整此变量，从而使得该变量是能够让函数模型误差最小的那个值，从而获得最优解。我认为通过其在线性回归的应用可以直观感受到。
3. 偏导数的核心应用是在梯度下降算法中。偏导数的运用能够不断地更新模型中的参数，分析参数的微小变化会对总误差的影响程度，总之就是用来衡量参数的影响程度。
### 链式法则
1. 链式法则是微积分定理中用来计算两个或多个函数组合而成的复合函数的导数。
2. 机器学习模型，尤其是从神经网络的学习中能够感受到，就是一系列函数的符合（在《深度学习的数学》一书对于多变量函数的链式法则的讲解中也是用了类似神经网络的图示）。它为导数与偏导数的复杂计算提供了数学原理与基础。
3. 
### 梯度
1. 梯度是个指向函数值增长最猛烈方向的向量。（类似于上山路上最陡峭但是最快的一条。）
2. 梯度由函数对所有变量的偏导数组成。梯度这个向量中的每个分量能够告诉我们函数在对应的坐标轴上的变化率。
3. 由于梯度是指向函数变化最快的方向，所以在机器学习中，“-梯度”就是指向函数变化最慢的方向。而恰恰在机器学习中就是要寻找使得函数模型的损失最少的函数，因此梯度很有用，它提供了明确的方向和幅度，指导着模型参数不断地调整与优化，有效减小误差。
### 矩阵
1. 在刚刚开始学习线性代数时，矩阵给我的最大感受就是这是一种更加简洁，高效的数学语言。矩阵通过根据问题构建模型，往往能够使得复杂问题变简单且直观。
2. 在机器学习的数据表示中，矩阵是很好的工具，能够让多个样本、多个特征融入在一个矩阵中。并且矩阵与神经网络有很大的关联，提供了一种能够高效执行的机器指令。
## 什么是“模型”？机器学习中的模型是如何工作的？
1. **什么是模型**
模型应该是一个能够模拟一些实际问题的函数（例如房价估计模型）或者是一种能够完成特定任务的工具，是通过实际探求出来的一种规律。
2. **机器学习中的模型是如何工作的**
   1. 机器学习中的模型是通过对与一些实例的分析与模拟而构建出的基于投入数据的一种“训练和预测”的模型。
   2. 通过不断地投入数据进行训练，数据和特征被模型不断地“学习”，最后也能够对输入的数据进行预测与输出。同时模型也会对投入的数据总结规律，进而进行更加合理的预测。
   3. 本质上就是上面的数学概念的宏观表现。通过梯度下降调整参数寻找最快的方向，同时通过链式法则来回退错误，迭代优化，加强规律的合理性，进一步去应用到其他的情况中去。

## 什么是监督学习？什么是无监督学习？请分别举一个例子。
1. 监督学习
- 监督学习是在对模型训练的过程中，给与数据的同时给予答案，让模型对其进行学习。
主要通过回归与分类学习，从有标签的训练数据中学习一个新的函数，建立新的关联，以便对新的数据做出预测。
- 举个栗子：在“房价预测”中，输入类似于房屋面积的数据，输出房价。
2. 无监督学习
- 无监督学习是要从无标签的数据中发现内在的结构或模式。
主要通过聚类将数据分为不同的组，找出共同点。模型是只给数据，不给答案，模型自行探索学习。
- 举个栗子：“客户细分”：根据购买行为将客户分为不同的群体，以便制定精准的营销策略。
虽然只输入了客户的购买行为，但模型会自行进行聚类，找出共同的部分，进行细分。
## AI 是什么？深度学习和传统机器学习的区别？
1. AI是什么？
- AI是一种让机器像人一样思考并解决问题的技术，是强大的人工智能，试图去解决在过往只能由人才能解决的问题。
- AI自身有学习能力与推理能力，能够自己“思考”，并去解决问题。
2. 深度学习和传统机器学习的区别？
   1. - 传统机器学习需要操作者本身去观察并分析数据，然后亲手设计、提炼出对预测目标最有效的特征。
      - 深度学习只需要操作者提供原始数据和最终目标。深度机器学习会自动从原始数据中提取特征，让提取特征的过程自动化了。
   2. - 传统机器学习对于数据的依赖度较低，而且推理时间短，适用于数据量较小的场景，易解释，仍是人来指导机器。
      - 深度学习高度依赖于大量的数据，有很高的性能。它需要很长的训练时间，可解释性较差。适用于数据量很大的场景中，并且在任务本身很复杂，难以用规则或手工来描述时，它具有着压倒性的优势，例如，自动驾驶，图像生成等。
## 怎么用矩阵乘法表示神经网络的全连接层前向传播过程？
 >  - 全连接层是神经网络中最基本的层之一，也被称为“稠密层”。它的核心思想是前一层的每一个神经元都与当前层的每一个神经元相连接。
 >  - 神经网络的全连接层的前向传播过程，本质上就是一系列的矩阵乘法和向量加法，再加上偏置项和激活函数的运用，从而得到结果。
1. 在从单个神经元到一层神经元的计算过程中，会用到`z = w ·x + b`的计算方式，若用简单基本的`for`循环来实现，会非常低效。而矩阵乘法可以让我们一次性并行地计算出整个层的所有输出。
2. **传播过程**
   1. 定义变量和维度
      > 假设网络结构如下：
      - 输入层：有n个特征。采用一个行向量表示一个输入样本：`X = [x1,x2,...,xn] `,其形状为(1，n)。
      - 全连接层有m个神经元。
      -   每个输入特征到每个神经元都有一个连接权重。因此，权重矩阵` W `的形状是 `(n, m)`。
      - 偏置： 每个神经元有一个偏置项。因此，偏置向量`b` 的形状是` (1, m)`。
      - 激活函数：用`σ`表示，如Sigmoid等。
   2. **矩阵乘法表示前向传播**
   `Z = X·W + b`
   `A = σ(Z)`
   - `z`是加权求和结果，`A`是该层的最终输出，他们两个的形状均是`(1,m)`。
   > - 其中的`X·W`用到了矩阵乘法，`X`的形状是 `(1, n)`,`W` 的形状是 `(n, m)`,由乘法规则可知结果， `X · W` 的形状是 `(1, m)`。
   > - 向量加法实质上是将偏置向量b加到结果上，得到z。

## Python 与 C 语言的区别？
1. **核心区别**
- **C语言**：是一门编译型、静态、弱类型的底层语言。它追求的是效率和灵活性，让程序员能够直接管理内存和代码，但底层细节较多。 

- **Python**:是一门解释型、动态、强类型的高级语言。它追求的是开发效率和可读性，语法简洁明了，拥有强大的标准库和第三方库，让开发者能快速实现想法，而无需关心底层细节。
2. **代码特点**：
- C 语言:
  - 需要包含头文件 #include <stdio.h>。
  - 函数 add 必须明确指定参数 a, b 和返回值都是int 类型。
  - 变量 num1, num2, result 都必须先声明为 int 类型。
  - 使用 printf 打印时需要指定格式 %d。
  - 需要先编译（gcc program.c -o program）才能运行（./program）。
- Python:
  - 语法极其简洁，没有分号、花括号，使用缩进定义代码块。
  - 函数 add 没有类型声明，a 和 b 可以是任何支持 + 操作的类型（整数、浮点数、字符串等），这体现了动态类型的特性。
  - 无需编译，直接通过解释器运行（python program.py）。
3.**应用场景**
- C 语言
  - 可以用来深入了解计算机科学原理（如内存管理、数据结构如何实现），打下坚实的编程基础，理解高级语言背后的机制。
  - 适用于操作系统、嵌入式开发、硬件驱动等需要极致性能或直接控制硬件的领域。
- Python
  - 适用于于数据科学、人工智能、Web 后端开发、自动化脚本等应用领域。

### 为什么需要 python 虚拟环境？
- 虚拟环境是 Python 开发中非常重要的工具，主要用于**隔离项目依赖**。
1. 虚拟环境可以解决依赖冲突
比如有两个项目A和B，需要两个不同的版本，因为全局只能有一个版本，所以后安装的会覆盖掉之前安装的，就会导致项目A和B产生冲突。
但如果创建了虚拟环境，那么A和B则可以分别在自己适合的虚拟环境版本下去进行，不会产生冲突。

2. 虚拟环境可以做到项目依赖隔离
如果有多个项目，在没有虚拟环境时，会出现包版本冲突、无法同时维护新旧项目、更新一个包可能破坏其他项目等此类问题，而虚拟环境则可以确保每个项目都有自己独立的 Python 包生态系统。

3. 虚拟环境可以避免污染系统python
虚拟环境中可以自由安装任何版本，不需要管理员权限，并且不影响系统其他部分。

4. 版本管理和重现性
能够让代码在本地正常运行的同时也能够在他人或服务器上运行正常。这一点可以保证团队协作的一致性。

## 什么是多层感知机（MLP）？其结构是怎样的？
### 什么是多层感知机（MLP）？

1. 多层感知机是一种经典的前馈人工神经网络。它的核心思想是通过在输入层和输出层之间加入一个或多个隐藏层，并引入非线性激活函数，使得网络能够学习并模拟复杂的非线性关系。

2. 感知机：
- **单层感知机**是最简单的形式，它只能解决线性可分的问题（例如，用一条直线就可以将两类数据分开），但其能力非常有限。

- **多层感知机**：当我们将多个感知机组装成层状结构，特别是加入隐藏层后，网络就具备了解决非线性问题的能力（例如，需要一个圆圈或更复杂的曲线才能分类的问题）。MLP是深度学习中最基础的构建模块之一。
### 其结构是怎样的？
- MLP的结构具有高度的规律性。
- 主要由以下部分组成：

1. 核心组成层

   1. 输入层：

   - 功能：负责接收原始数据。这一层不进行任何计算，仅代表输入的特征。

   - 神经元数量：等于输入数据的特征数量。

   2. 隐藏层：

   - 功能：这是MLP的“大脑”，是进行信息处理和特征提取的核心部分。之所以称为“隐藏”，是因为这些层在训练数据中是不可见的。

   - 结构与数量：一个MLP至少有一个隐藏层，也可以有多个。每个隐藏层都包含若干个神经元。深层网络（多个隐藏层）能够学习更抽象、更复杂的特征。

   - 核心操作：每个隐藏层神经元都会对其所有输入进行“加权求和 → 加偏置 → 通过激活函数”的操作。

   3. 输出层：

   - 功能：产生网络的最终预测结果。

   - 神经元数量：由任务类型决定。

     - 二分类：通常1个神经元（使用Sigmoid激活函数，输出一个0到1之间的概率值）。

     - 多分类：神经元数量等于类别数（使用Softmax激活函数，输出每个类别的概率）。

     - 回归：通常1个神经元（使用线性激活函数，输出一个连续值）。
2. 多层感知机具有全连接和前馈等关键结构特性。
   - 全连接：
    MLP最显著的特征是全连接，意味着某一层的每一个神经元都与下一层的每一个神经元相连接。因此，它也被称为全连接网络。

   - 前馈：
    数据只能从输入层流向隐藏层，再流向输出层，方向是单向的，没有循环或反馈。这与循环神经网络（RNN）不同。

## 数据在神经网络中扮演哪些角色？（数据集的 split 和处理）

1. 数据集的split

> 数据集的split操作是一种常用的数据处理技术，用于将数据集或张量按照指定方式分割成多个小块。这种操作在数据预处理、模型训练和特征提取等环节中具有重要作用，能够有效提高计算效率和模型性能。

- 为了有效利用数据，我们通常会对其进行分割和处理,而且通常将一个完整的数据集分成三个互不重叠的部分：

1. 训练集：
   训练集是模型用来学习的**教材**和数据。模型通过分析训练集中的样本和其对应的标签，来调整内部参数（权重和偏置），从而学会如何从输入数据映射到输出结果。【它通常是占比最大的部分】

2. 验证集：
   - 验证集相当于在训练过程中**对模型阶段性学习效果的验证**。
   - 它并不参与模型参数的学习，而是用来：
      1. **调整超参数**（如学习率等）
      2. **监控是否过拟合**（如果训练集损失持续下降，但验证集损失开始上升，说明模型过拟合了）。
      >注：**过拟合**指的是只能拟合训练数据, 但不能很好地拟合不包含在训练数据中的其他数据的状态。
      3. **进行早停**。
      >注：早停法可以降低每个参数维度的有效规模，进而解决过拟合问题。
   - 它占比较小。

3. 测试集：
   - 测试集相当于**最后的一次评估**。（有点像高考的感觉 `@_@` ）。在模型训练和调参全部完成后，我们用测试集对模型进行最终、一次性的评估。测试集在整个过程中必须完全“隔离”，绝不能以任何形式参与训练或调参。
   - 测试集能够用来评估模型的泛化能力。
   >注；**泛化能力**指的是一个模型对于在训练过程中未曾见过的数据的表现能力。也就是说，一个具有良好泛化能力的模型在训练集之外的数据上也能表现出色。
   - 通常也占比较小。
- 综上所述，数据集的split的核心目的是为了评估模型的泛化能力。也就是评估一个模型的举一反三的能力，这一点很像我们人类学习，都需要在训练中把握精髓，进而应用在新的方面，不断进步。

2. 处理
- 数据集在扔进网络前需要进行处理，不能直接扔进去，通常进行以下几种处理方式：
   1. **归一化/标准化**：将数据缩放到一个特定的范围（如 [0,1]）或分布（均值为0，方差为1）。这可以加速模型收敛，提高训练稳定性。

   2. **数据增强**：对训练集数据进行随机变换（如旋转、裁剪、翻转图像），在不增加新数据的前提下，有效增加数据多样性，是防止过拟合的强有力手段。

   3. **处理缺失值**：填充或删除数据中的缺失值。

### 噪声是什么？特征是什么？标签是什么？

#### 噪声

   1. 定义：数据中与最终任务无关的、随机的干扰信息或误差。

   2. 来源：

      1. 数据采集：传感器误差、图像中的杂点。

      2. 数据标注：人为标注错误。

      3. 数据本身：无关的背景信息（如在识别鸟的照片中，背景里的树木和天空如果变化多端，对于“识别鸟”这个任务来说也是一种噪声）。

   3. 影响：噪声会干扰模型学习真正的、有意义的规律，可能导致模型性能下降和过拟合。一个好的模型应该学会忽略噪声，抓住本质特征。

#### 特征
   1. 定义：输入数据中用于预测或描述的属性或变量。它是模型的输入。

      > 例子：

      - 在房价预测中，特征可以是：房屋面积、卧室数量、地理位置、房龄等。

      - 在图像识别中，特征就是图像中的每一个像素值。

   2. 角色：特征是模型做出决策的依据。

#### 标签
   1. 定义：我们想要模型预测的正确答案或目标输出。在监督学习中，每个样本都对应一个标签。

   > 例子：

   - 在房价预测中，标签就是房子的真实价格。

   - 在猫狗分类中，标签就是“猫”或“狗”。

  2. 角色：标签是模型学习的目标。通过比较模型的预测输出和真实标签，我们可以计算损失，从而驱动模型进行学习。


### Batch size 是什么？为什么堆叠成 Batch 可以提高运算速度？

#### Batch size 是什么？
- Batch size 是指在深度学习模型训练过程中，每次迭代所使用的样本数量。换句话说，它决定了模型在更新权重时所使用的数据量大小，它的大小直接影响了模型的优化过程。
- 作用：在深度学习中，我们通常使用大量的数据来训练模型。如果每次迭代都使用整个数据集，就会出现计算量大，收敛速度慢的问题。所以引入了Batch size的概念，将数据集分成多个小批量，每次迭代只使用一个小批量来更新权重。
- 这样就叫作小批量梯度下降，它具有计算量小，收敛速度快，泛化能力强的优点。

#### 为什么堆叠成 Batch 可以提高运算速度？
- 堆叠成 Batch就是进行批量处理，如果不进行堆叠就是一个一个进行处理，那么显然可以提高效率，因为堆叠成 Batch可以完美契合现代硬件的“并行计算”架构。
- 技术上的核心原因：并行化的矩阵运算
> 神经网络底层的所有计算（前向传播、反向传播）本质上都是矩阵乘法。而GPU正是为大规模并行矩阵运算而生的。
1. 从“向量”到“矩阵”的质变
   1. 单个样本：输入数据是一个向量。这个向量与权重矩阵W相乘，得到输出。这是一个 `向量 · 矩阵` 的运算。计算量小，且基本上是串行的，无法有效利用GPU的数千个核心。
   2. 一个Batch的样本：输入数据是 `[batch_size, 特征数]` 的一个矩阵。这个矩阵与同一个权重矩阵`W`相乘，其结果是一次性同时得到整个Batch所有样本的输出。这是一个 `矩阵 · 矩阵` 的运算。
2. 硬件的利用（GPU）
   GPU拥有成千上万个小型、高效的计算核心。当它执行一个大的矩阵乘法时，可以将这个巨大的计算任务分解成无数个小任务，然后分配给所有这些核心同时进行计算，进而就可以加速`矩阵 · 矩阵`运算。
3. 内存访问
一次性读取一个大的、连续的数据块（一个Batch），比反复读取大量的小数据块（单个样本）要高效得多。这减少了内存访问的延迟和开销。

- 堆叠成Batch，就是把许多个小的、串行的计算任务，打包成一个大的、并行的矩阵运算任务，从而让GPU能全力发挥其并行计算的优势，最终实现运算速度的成倍提升。

### 神经元是什么？
1. 从生物角度来说：
- 生物神经元是大脑和神经系统中的基本信号处理单元。

   1. 构成：在一个个的神经中枢中，神经元彼此相连，主要由树突、胞体、轴突构成。
   2. 工作原理：
   - 集成：树突从成千上万个其他神经元接收信号。这些信号有兴奋性的和抑制性的。
   - 触发：胞体将所有输入信号进行累加。如果累加的总和超过了一个特定的阈值，神经元就会被“激活”，产生一个自身的电信号，称为动作电位。
   - 传播：这个动作电位沿着轴突传播下去。
   - 传递：当电信号到达轴突末端的突触时，会释放神经递质，将信号跨过突触间隙，传递给下一个神经元的树突。
   3. 核心特征：
   - “全有或全无”：动作电位要么发生，要么不发生，但其发生的频率可以传递信息的强度。（这一点也在神经网络中被借鉴。）

2. 从深度学习角度来说：
- 深度学习中的神经元是对生物神经元的极度简化和数学抽象。它是构成人工神经网络的基本构建块。
- 它的工作流程模拟了生物上的神经元；
   1. 加权求和：
   - 每个输入连接都有一个权重，这模拟了生物突触的连接强度。重要的输入拥有更大的权重。神经元将所有输入乘以各自的权重，然后加上一个偏置项。偏置类似于生物神经元的阈值，它控制着神经元被激活的难易程度。

   2. 激活函数：
   - 激活函数为神经元引入了非线性，这是神经网络能够学习复杂模式的关键。
   
3. - 深度学习中的神经元就是类比了生物的神经元。
   1. 输入——树突接收到的信号。
   2. 权重——突触的连接强度。
   3. 加权求和 & 偏置——细胞体整合信号并与阈值比较。
   4. 决定是否产生动作电位的“全有或全无”机制（但现代的激活函数是连续的。）

### 什么是计算图？它和数据结构/离散数学中学的图有什么区别？怎么构建计算图？
#### 计算图
1. 
- 计算图（Computation Graph）是深度学习中用来表示数据流和操作的图形化结构。
- 计算图主要有两种类型：静态计算图和动态计算图。

2. 组成：
   1. 节点

   计算图中的每个节点表示一个数学操作或一个数据元素。
   - 操作节点：表示加法、乘法、矩阵乘法、激活函数等。
   - 变量节点：表示神经网络中的变量或参数（如权重、偏置），这些变量在计算过程中会被更新。

   2. 边
   - 计算图中的每条边表示数据（张量）的流动。边连接操作节点和变量节点，表示数据如何在网络中传播。

   3. 输入与输出：
   - 输入节点：模型输入的数据（如图像、文本）。
   - 输出节点：模型的最终结果，如分类标签、预测值。
- 它通过将数学表达式转化为图的形式来帮助描述神经网络中的前向传播和反向传播过程。

#### 计算图和数据结构/离散数学中学的图的区别
- 计算图关心的是输入数据经过操作节点，按照一定的顺序最终得到输出。
- 数据结构/离散数学中的图关心的是节点和边的连接关系和拓扑结构。

1. 目的；
- 计算图表示和执行计算，核心是为了自动微分以优化模型参数。它是一个工作流程。
- 数据结构/离散数学中学的图表示实体间的抽象关系，用于分析结构、寻找路径、优化网络。它是一个关系网。

2. 语义：
   1. 节点：

   - 计算图：数据或操作。如叶子节点：输入数据、模型参数（常量/变量）。或是中间节点：数学运算（如 +, ×, ReLU）。

   - 数据结构/离散数学中的图：实体或对象。 如：城市、人物、网页、网络路由器。

   2. 边；

   - 计算图：数据依赖关系和计算流向。边传递的是具体的数值（张量），方向表示“计算顺序”。

   - 数据结构/离散数学中的图：实体间的关联。边可能有权重（如距离、成本、关系强度），方向表示“关系方向”（如A关注B）。

3. 关键结构特性：
- 计算图：几乎总是“有向无环图”。因为计算是分步进行的，不能有循环依赖（否则无法计算）。动态图允许控制流，但其底层表示在单次前向传播中仍是DAG。

- 数据结构/离散数学中的图：结构灵活多样。有向/无向：均可。带环/无环：均可（社交网络必有环）。

4. 核心算法与应用：

- 计算图：
   - 前向传播：计算输出值。
   - 反向传播：基于链式法则，从输出反向计算所有参数的梯度。这是其存在的首要意义。

- 数据结构/离散数学中的图：
   - 路径搜索：Dijkstra算法，A*算法。
   - 遍历：BFS，DFS。
   - 连通性：判断两个节点是否相连。
   - 社区发现、排序：PageRank算法等。

### 什么是隐藏层（hidden layers）？它为什么叫这个名字？

#### 隐藏层
- 在一个神经网络中，隐藏层是介于输入层和输出层之间的那些层。
- 隐藏层是神经网络的“大脑”和“引擎”。他们会逐层进行特征提取与转换，并且可以增加非线性表达能力，正是通过这些隐藏层中的非线性变换，神经网络才能学习并模拟现实世界中复杂的、非线性的关系，而不仅仅是简单的线性拟合。

#### 原因

1. 对于外部观察者而言，它是“看不见的”，数据在经过这些中间层时是如何被转换、组合和抽象的是完全不透明和不可直接解释的。这些层就像一个黑箱，被“隐藏”在了网络的内部。

2. 与生物神经网络的类比

- “中间神经元”同样是无法直接从外部观察到的，它们隐藏在大脑皮层之下。因此，人工神经网络中的对应层就被命名为“隐藏层”。

### 什么是损失函数？什么任务用什么损失函数？

#### 损失函数

- 深度学习中的损失函数是一个衡量预测结果与真实结果之间差异的函数 ，也称为误差函数。它通过计算模型的预测值与真实值之间的不一致程度，来评估模型的性能。

#### 什么任务用什么损失函数？

- 损失函数按任务类型分为**回归损失**和**分类损失**。

1. 回归损失主要处理**连续型变量**，常用MSE、MAE等，对异常值敏感度不同。

   1. 均方误差（MSE）
   
   - 公式： `(1/n) * Σ(真实值 - 预测值)²`

   - 特点： 对异常值（离群点）非常敏感，因为误差被平方了，大的误差会被放大得更大。

   - 使用场景： 大多数回归问题的默认选择，当数据中异常值不多时效果很好。

   2. 平均绝对误差（MAE）
   
   - 公式： `(1/n) * Σ|真实值 - 预测值|`

   - 特点： 对异常值不那么敏感，因为它只计算绝对距离。

   - 使用场景： 当数据中存在明显异常值时，MAE比MSE更稳健。

   3. Huber Loss

   - 公式： 结合了MSE和MAE。当误差较小时，它像MSE（二次）；当误差较大时，它像MAE（线性）。

   - 特点： MSE和MAE的折中方案。既对异常值没有MSE那么敏感，又在误差小的地方保持了平滑的优化特性（这对梯度下降有利）。

   - 使用场景： 当你觉得数据有异常值，但又不想完全像MAE那样忽略它们时使用。

- 回归损失是损失函数在回归问题中的具体应用。回归问题是指预测一个或多个连续值的问题，与分类问题（预测离散值）相对。

2. 分类损失主要处理离散型变量，常用Cross Entropy Loss、Dice Loss等，适用于不同分类任务需求。

   1. 交叉熵损失

   - 公式： -Σ(真实标签 * log(预测概率))

   - 思想： 它衡量的是模型输出的概率分布与真实概率分布之间的差异。

   - 二分类交叉熵损失： 用于只有两个类别的问题（如正面/负面）。

   - 多分类交叉熵损失： 用于两个以上类别的问题。这是分类任务中最常用、最有效的损失函数。

   - 使用场景： 几乎所有的分类模型，包括逻辑回归和神经网络。

   2. Hinge Loss

   - 公式： max(0, 1 - 真实标签 * 预测值)

   - 特点： 旨在不仅要求分类正确，还要求有足够大的“置信度边界”。

   - 使用场景： 主要用于支持向量机（SVM）。

- 分类损失是在训练分类模型时，用于衡量模型预测结果与真实标签之间差异的一种度量。它是一个非负值，反映了模型预测结果的准确性。分类损失越小，意味着模型的预测结果与真实标签越接近，模型的性能也就越好。

### 前向传播是什么？梯度是什么？学习率是什么？反向传播是什么？有哪些常见的优化器？

#### 前向传播

- 前向传播指的是数据从神经网络的输入层开始，经过一系列隐藏层的计算，最终到达输出层，并得到预测结果的过程。整个过程是单向的，从输入到输出，没有反馈循环。

#### 梯度
- 梯度是一个向量，箭头的方向指向该点处最陡峭的上升方向。箭头的长度（模） 表示这个斜坡的陡峭程度。

- 在数学上，梯度是多变量函数的所有偏导数构成的向量。

#### 学习率

- 学习率是深度学习和机器学习中的一个关键超参数，它决定了模型在训练过程中更新权重参数的速度和方向。

- 梯度告诉你下山的方向，学习率则决定了你每一步迈多大。

#### 反向传播

- 计算梯度的顺序与前向传播完全相反。前向传播是从输入层到输出层，而反向传播是从输出层开始，将误差（梯度）一层一层地反向传递回前面的层。

- 反向传播的核心优势在于它极其高效。它利用链式法则，通过一次前向传播和一次反向传播，就能一次性计算出损失函数对所有参数的梯度，其计算复杂度大致相当于两次前向传播。

#### 常见的优化器

1. SGD (随机梯度下降)
- 这是最基础、最经典的优化器。
- 核心思想：直接沿着当前批次（Batch）数据计算出的负梯度方向更新参数。

2. SGD with Momentum (带动量的SGD)
- 为了解决SGD的震荡问题，引入了物理中“动量”的概念。

- 核心思想：不仅考虑当前的梯度，还保留之前更新方向的“惯性”。这有助于在相关方向上加速学习，并抑制震荡。

3. NAG (Nesterov Accelerated Gradient)
- NAG是Momentum的一个“有远见”的变体。

- 核心思想：在Momentum中，我们先用当前动量走一步，再计算梯度。而NAG是先按照动量走一步，然后在那一点计算梯度，并用这个“向前看”的梯度来修正当前的更新。

4. AdaGrad (Adaptive Gradient Algorithm)
- 之前的优化器对所有参数都使用同一个学习率。AdaGrad打破了这一点。

- 核心思想：为每一个参数自适应地调整学习率。对于频繁更新的参数（其梯度平方累积值大），我们给予较小的学习率；对于不频繁更新的参数（其梯度平方累积值小），我们给予较大的学习率。这非常适合处理稀疏数据。

5. RMSprop (Root Mean Square Propagation)
- 为了解决AdaGrad学习率急剧下降的问题，RMSprop应运而生。

核心思想：对AdaGrad的梯度平方累积项进行指数加权移动平均，而不是简单求和。这样，`G_t`就是一个“泄漏”的累积，更关注最近的梯度 history，从而避免了学习率无限变小的问题。

6. Adam (Adaptive Moment Estimation)
- 目前最流行、最常用的优化器。它结合了Momentum（一阶矩） 和 RMSprop（二阶矩） 的优点。

- 核心思想：
   1. 计算梯度的一阶矩（均值）`m_t`的指数加权平均，类似于Momentum。

   2. 计算梯度的二阶矩（未中心化的方差）`v_t`的指数加权平均，类似于RMSprop。

   3. 对一阶矩和二阶矩进行偏差校正，以解决它们在初始时刻的偏差问题。


### 归一化是什么？正则化是什么？

#### 归一化

1. 将一列数据变化到某个固定区间(范围)中，通常，这个区间是[0, 1] 或者（-1,1）之间的小数。这样可以让数据处理更加便捷快速。
2. 归一化是一种简化计算的方式，即将有量纲的表达式，经过变换，化为无量纲的表达式，成为纯量。

#### 正则化

正则化就是规范化，正则化就是说给损失函数加上一些限制，通过这种规则去规范他们再接下来的循环迭代中，不要自我膨胀。正则化是用来防止模型过拟合而采取的手段。

### 什么是欠拟合？什么是过拟合？

#### 欠拟合

欠拟合：泛化能力差，训练样本集准确率低，测试样本集准确率低。

#### 过拟合

过拟合：泛化能力差，训练样本集准确率高，测试样本集准确率低。

> (合适的拟合程度：泛化能力强，训练样本集准确率高，测试样本集准确率高)

#### 在预测房价的模型中，模型的输出是什么？它是连续的还是离散的？

2. 模型的输出是连续的。此问题是个回归问题，回归问题的输出是连续值。

#### 什么是“线性回归”？试着用一句话描述它是如何工作的。解释你在高中学习的“最小二乘法”的原理。

1. 线性回归通过寻找一条=最佳拟合直线，使得所有数据点到这条直线的预测值与实际值之差的平方和最小，从而来描述一个或多个自变量与一个连续因变量之间的关系。

2. 
- 最小二乘法是线性回归中用于找到那条“最佳拟合直线”的具体方法。

- 
   1. 例如要找到一条直线 y = ax + b（其中 a 是斜率，b 是截距），使得这条直线在所有可能的直线中“最好”地穿过数据点。最小二乘法的核心，就是通过数学方法找到一组 `a` 和 `b` 的数值，使得这个“误差平方和”达到全局最小。(这个“误差平方和”用公式表示就是：`Loss = Σ(y_i - ŷ_i)² = Σ(y_i - (a*x_i + b))²`)
   2. 求解过程：我们可以把 Loss 看作是一个关于 a 和 b 的二元函数。为了求这个函数的最小值，我们分别对 a 和 b 求偏导数，并令导数等于零。这样就得到了一个关于 a 和 b 的二元一次方程组（称为正规方程）。解这个方程组，就能得到唯一的、能使误差平方和最小的 a 和 b 的值。

- 总结最小二乘法的原理：通过最小化所有数据点的预测值与真实值之间差值的平方和，来求解线性回归模型中的最优参数。

#### 评价一个回归模型的好坏，常用的一个指标有什么？

- 最常用的核心指标：均方误差，简称 MSE。

- MSE 计算的是模型的预测值与真实值之间差异的平方的平均值。

- 公式：MSE = (1/n) * Σ(真实值 - 预测值)²

## 分类

### 概念理解

#### 分类任务的输出是什么？它和回归任务的输出有什么本质不同？

1. 分类任务的输出是离散的类别标签，有以下几种类型：
   - 二分类任务：输出只有两个类别标签，例如“是”或“否”、“健康”或“不健康”。
   - 多分类任务：输出包含两个以上的类别标签，例如手写数字识别（0到9）、植物种类分类等。
   - 多标签分类任务：每个样本可能属于多个类别，输出是多个标签的组合。

2. 本质不同：
- 回归任务的输出是连续的，是对于有无限多种可能的问题，去训练出一个模型，传入参数后再得到一个确定的数。正是因为我们所预测的变量有无限多种可能，在数轴上是连续的，所以称为连续变量。

- 分类任务的输出是离散的。分类任务本身就是要去做出是否与概率大小的判断，这样的结果只有几个值或多个值，就可以把每个值当作一类，去预测是哪一类，是离散的。
