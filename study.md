4.  !x
    这是对变量x进行取反操作。!是逻辑非运算符，用于将一个条件的结果取反。

    如果x的值为非零（即x为真），那么!x的值将为零（即假）。

    反之，如果x的值为零（即x为假），那么!x的值将为非零（即真）。

    !x != 0表达式的含义是：当x的值为零时，该表达式的值为真；当x的值不为零时，该表达式的值为假。

6. for (初始化表达式; 条件表达式; 更新表达式) {
    // 循环体
    //也就是要重复执行的代码
}
它的运行过程为：
1) 先执行初始化表达式。

2) 再执行条件表达式，判断条件是否成立：如果表达式的值为真（非零），则执行循环体；如果表达式的值为假（零），则结束循环（跳出循环），继续执行整个 for 循环后面的代码。

3) 执行完循环体后再执行更新表达式。

4) 重复执行步骤 2 和 3，直到条件表达式的值为假，就结束循环。

for(i=0;i<10;i++)
与
for(i=0;i++<10)
不同
因为第二个的i++提前进入了循环体

%o：八进制输出 → 125 的八进制是 175

9. 在 C 语言中，浮点数字面量的规则是：

必须有数字（整数或小数部分，或指数部分配合数字）。

指数符号 e 或 E 后面必须是整数（可带正负号）。

[数字部分] e [指数部分]
其中：

数字部分：可以是整数（如 3）或小数（如 2.33）。

e（或 E）表示“乘以 10 的…次方”。

指数部分：必须是整数，可带正负号（如 -1、+2）。

例子：
2.33e-1 表示
2.33×10−1=0.233

10e2 表示 
10×102=1000

在 C 语言中：
e 前后都必须有数字（但数字部分可以只有小数点一边的数字，如 .5e2 或 3.e1 也可以）。
